# 起个变量名

一款便捷的变量命名工具，基于百度翻译 API，将中文快速翻译成英文，并自动生成多种编程规范的变量命名格式。

## 功能特点

- 支持中译英
- 自动生成 10 种变量命名格式
- 一键复制，点击结果即可复制并退出
- 两种翻译模式可选

## 支持的命名格式

| 格式名称 | 示例      |
| -------- | --------- |
| 小驼峰   | userName  |
| 大驼峰   | UserName  |
| 横线分隔 | user-name |
| 下划线   | user_name |
| 常量格式 | USER_NAME |
| 句子格式 | User name |
| 全小写   | username  |
| 全大写   | USERNAME  |
| 点分隔   | user.name |
| 路径分隔 | user/name |

## 使用方法

### 1. 配置翻译服务

首次使用时，需要配置百度翻译 API：

1. 点击右上角设置按钮
2. 前往 [百度翻译开放平台]`https://fanyi-api.baidu.com/manage/developer` 注册并申请翻译服务
3. 填写 APP ID 和密钥
4. 配置将自动保存

### 2. 输入翻译内容

在输入框中输入需要翻译的中文内容。

### 3. 选择翻译模式

- **点击/回车翻译**：输入完成后点击"翻译"按钮或按回车键
- **实时翻译**：输入后自动翻译，可通过滑块调整延迟时间

### 4. 复制结果

点击喜欢的命名格式，结果将自动复制到剪贴板，并关闭插件。

## 注意事项

- 如果翻译时直接弹出设置面板，说明 APP ID 和密钥未配置或配置有误
- 请确保 APP ID 和密钥配置正确，否则无法使用翻译功能
- 百度翻译 API 有额度限制,个人认证-100万字符/月，企业认证-200万字符/月，具体参考官方文档
- 实时翻译模式会在每次输入后自动执行，适合快速连续输入场景
