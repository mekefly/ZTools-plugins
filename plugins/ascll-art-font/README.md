# ASCII Art Font

> ASCII 艺术字体生成器 - ZTools 插件

基于 [figlet.js](https://github.com/patorjk/figlet.js) 实现，输入文字即刻转换为酷炫的 ASCII 艺术字。内置 329 种字体，支持收藏、配色切换和布局调节。

## 功能特性

- **329 种字体** - 内置全部 figlet 字体，涵盖 3D、手写、像素、哥特等多种风格
- **实时预览** - 输入文字即时生成 ASCII 艺术字
- **8 种配色方案** - 白色、灰色、暖色、蓝色、深蓝、紫色、深色、黑色主题
- **字体收藏** - 收藏常用字体，快速筛选
- **字体翻页** - 左右箭头快速浏览切换字体
- **一键复制** - 将生成结果复制到剪贴板
- **布局设置** - 可调节水平/垂直间距（默认、完整、紧凑、控制挤压、通用挤压）
- **设置持久化** - 收藏、配色、字体选择等设置自动保存

## 使用方式

在 ZTools 搜索栏输入以下关键词启动插件：

- `ascii`
- `艺术字`
- `art font`
- `figlet`

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 技术栈

- Vue 3 + TypeScript + Vite
- figlet.js（ASCII 艺术字生成引擎）
