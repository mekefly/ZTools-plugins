# 2FA TOTP 插件 (2fa-totp)

## 简介
这是一个为 ZTools 平台开发的 2FA TOTP 动态验证码管理插件。它提供安全、便捷的双重认证验证码管理功能，支持置顶、排序、实时预览以及自定义 UI 样式。

## 特性
- **实时预览**：倒计时结束时自动更新验证码。
- **自定义样式**：支持环形与条形两种时间指示器。
- **灵活管理**：支持账号置顶、排序及右键菜单操作。
- **快捷复制**：点击验证码自动复制，并支持自动回填。
- **视觉反馈**：最后 5 秒渐变预警，提醒用户验证码即将过期。
- **外部跳转**：关于页面的链接直接通过系统默认浏览器打开。
- **性能优化**：基于 Vue.js 与原生 JS，轻量、流畅。

## 安装与开发
- 依赖项：仅需加载 `vue.global.js`。
- 入口文件：`index.html`。
- 逻辑层：`app.js`。
- 预加载：`preload.js`。
