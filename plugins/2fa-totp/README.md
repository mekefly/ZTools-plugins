# 2FA TOTP 插件 (2fa-totp)

## 简介
这是一个为 ZTools 平台开发的 2FA 动态验证码管理插件。它提供安全、极速、美观的多协议双重认证管理功能。

## 特性
- **全协议支持**：完美支持 **TOTP** (RFC 6238)、**HOTP** (RFC 4226) 以及 **Steam 令牌** (自定义 Base26 算法)。
- **高级配置**：
    - 支持多种哈希算法：SHA1, SHA256, SHA512。
    - 支持自定义验证码位数：5 (Steam), 6, 7, 8 位。
    - 支持自定义时间周期 (15/30/60s) 或手动计数器。
- **极致交互**：
    - **下一个验证码预览**：支持在主界面实时显示下一个周期的验证码（可配置开关）。
    - **智能联动**：在选择 Steam 协议时，自动锁定并填充协议规范（SHA1/5位/30s）。
    - **自动清洗**：密钥输入时自动转换为大写并去除空格，提升录入体验。
    - **HOTP 5s 节流刷新**：专为计数型协议设计的带冷却保护的刷新按钮。
    - **视觉反馈**：全域点击快捷复制，复制成功后 2s 勾号提示。
- **现代 UI**：网格化配置界面，所有垂直菜单均采用 Drop-up 上拉模式适配小窗；支持指示器模式切换（环形/条形）。

## 安装与开发
- 依赖项：仅需加载 `vue.global.js`。
- 入口文件：`index.html`。
- 逻辑层：`app.js`。
- 预加载：`preload.js`。
