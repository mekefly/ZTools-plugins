# 2FA 动态验证码

> 这是一个为 ZTools 平台开发的 2FA 动态验证码插件，提供安全、简约且高效的 2FA 令牌管理器，支持 TOTP/HOTP/Steam 多种协议，基于 Vue 3 + Vite + TypeScript 构建。

## ✨ 核心特性

- **多协议适配**：完美支持 TOTP 时间型、HOTP 计数型以及 Steam 令牌专用 Base26 算法。
- **工业级安全**：采用 PBKDF2 实施工业级密钥派生，结合 AES-GCM 算法对存储数据进行加密保护。
- **硬件级绑定**：支持设备指纹绑定，实现开启即自动解锁 TOTP 生成。
- **隐私防护**：查看或编辑密钥原文强制进行身份验证，支持会话期免密查看。
- **极致体验**：支持账号置顶、拖拽排序、实时预览下一周期验证码及多种进度指示器样式。

```text
├── public/
│   ├── logo.png              # 插件图标
│   ├── plugin.json           # 插件配置文件
│   ├── README.md             # 插件市场展示说明
│   └── preload/              # Preload 脚本目录
├── src/
│   ├── components/           # UI 组件库
│   │   ├── Modals/           # 模态框组件

│   │   ├── useAccounts.ts    # 账户增删改查与解密
│   │   ├── useAuth.ts        # 身份验证及密钥派生
│   │   └── useTicker.ts      # 高精度计时器与令牌更新
│   ├── utils/                # 工具函数
│   │   ├── crypto.ts         # 加密与安全存储集成
│   │   └── otp.ts            # OTP 核心算法实现
│   ├── App.vue               # 插件主入口
│   ├── constants.ts          # 全局常量定义
│   ├── main.css              # 全局样式系统
│   └── main.ts               # 项目启动引导
├── vite.config.js            # Vite 构建配置
└── tsconfig.json             # TypeScript 类型配置
```
