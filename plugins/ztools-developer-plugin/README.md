# ZTools 开发者工具

专为 ZTools 插件开发者打造的工作台，提供开发中插件的统一管理与调试能力。

## 功能介绍

### 开发工作台

在工作台中集中管理所有注册为「开发模式」的插件项目。

- **插件列表**：左侧展示当前设备已绑定的开发中插件，支持快速切换
- **概览面板**：查看插件基本信息、版本、路径、本地绑定状态
- **快捷操作**：在文件管理器中定位插件目录、安装/卸载开发模式、重载插件

### 快捷指令

| 指令 | 说明 |
|------|------|
| `开发者工具` | 打开开发者工作台主界面 |
| `重载开发中插件` | 一键重载所有已安装开发模式的插件 |
| 拖入 `plugin.json` | 快速将插件项目添加到工作台 |

## 使用方式

1. 在 ZTools 搜索框输入「开发者工具」打开工作台
2. 点击左侧「+ 添加插件」，选择插件项目的 `plugin.json` 完成注册
3. 选中插件后可在右侧面板查看详情并执行相关操作
