# IP Switcher - 电脑 IP 便捷切换工具

#### 介绍

> 【uTools插件】快捷修改本机 IP 地址，保存多套 IP 方案，一键切换。
>
> 项目使用 `react` 开发，UI 采用 `ant-design`。

#### 注意事项

1. 当前工具暂时仅支持 Windows 系统，敬请关注后续更新。
