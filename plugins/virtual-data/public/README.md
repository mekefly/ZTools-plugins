## 📖 使用指南

### 1. 数据配置

1. **进入数据配置页面**：打开插件后，点击「数据配置」标签页
2. **添加数据模板**：
   - 点击「添加数据」按钮
   - 填写「数据名称」和「执行代码」
   - 点击「保存」按钮

3. **编辑数据模板**：
   - 点击数据项右侧的「编辑」按钮
   - 修改名称或代码
   - 点击「保存」按钮

4. **删除数据模板**：
   - 点击数据项右侧的「删除」按钮
   - 确认删除操作

5. **调整顺序**：
   - 拖动数据项左侧的拖拽图标来调整顺序
   - 排序会自动保存到数据库

### 2. 使用虚拟数据

1. **进入虚拟数据页面**：点击「虚拟数据」标签页

2. **选择显示模式**：
   - **卡片模式**：以卡片形式展示数据
   - **列表模式**：以列表形式展示数据

3. **复制数据**：
   - **卡片模式**：点击任意卡片
   - **列表模式**：点击任意列表项
   - 数据会自动复制到剪贴板

4. **搜索数据**：
   - 在搜索框中输入关键词
   - 系统会实时筛选匹配的数据

### 3. 系统设置

1. **进入设置页面**：点击「系统设置」标签页

2. **调整设置**：
   - **点击复制**：开启后，点击数据项会自动复制
   - **复制后提示**：开启后，复制成功会显示提示
   - **复制后关闭**：开启后，复制成功会自动关闭窗口

3. **设置会自动保存**，下次打开插件时会保持上次的设置

## 🎯 示例数据模板

### 基本信息

- **中文姓名**：`Mock.Random.cname()`
- **邮箱地址**：`Mock.Random.email()`
- **手机号码**：`Mock.Random.phone()`
- **身份证号**：`Mock.Random.id()`
- **地址**：`Mock.Random.county(true)`

### 业务数据

- **订单号**：`'ORD' + Mock.Random.string('number', 8)`
- **产品名称**：`Mock.Random.pick(['手机', '电脑', '平板', '耳机', '手表'])`
- **价格**：`Mock.Random.float(100, 10000, 2, 2)`
- **状态**：`Mock.Random.pick(['待处理', '处理中', '已完成', '已取消'])`

### 复杂数据

- **用户信息**：`{name: Mock.Random.cname(), age: Mock.Random.integer(18, 60), email: Mock.Random.email()}`
- **商品列表**：`Array.from({length: 3}, () => ({name: Mock.Random.word(), price: Mock.Random.float(100, 1000, 2, 2)}))`

## ❓ 常见问题

### Q: 如何添加新的数据模板？

A: 进入「数据配置」标签页，点击「添加数据」按钮，填写名称和执行代码，然后保存。

### Q: 如何修改数据模板的顺序？

A: 在「数据配置」标签页中，拖动数据项左侧的拖拽图标来调整顺序。

### Q: 复制功能不工作怎么办？

A: 检查「系统设置」中的「点击复制」选项是否开启，确保浏览器权限允许复制操作。

### Q: 如何使用复杂的 Mock.js 语法？

A: 参考 [Mock.js 官方文档](http://mockjs.com/)，使用完整的 Mock.js 语法来定义数据生成规则。
